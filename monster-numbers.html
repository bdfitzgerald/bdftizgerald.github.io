Project Details

Monster Numbers is a voice based app the Google Assistant. It is an interactive mental maths based quiz game aimed towards children to help them improve their mental maths skills. I used dialogflow.com to create a conversational interface, then by using a cloud function hosted firebase I created a response depending on the inputs using a webhook. Currently Monster Numbers can ask individual questions or a series of questions based on multiplication division subtraction or addition. The difficulty level can be also be changed between easy medium and hard to suit the user.
