Project Details

Monzo Web is a project i started based on Monzo, which is a new bank that runs exclusively on mobiles. They also have an open API which can be used to develop your own apps with their information. With this in mind I wanted to create a web based alternative for easier tracking of my spending. Monzo automatically categorises your transactions as well as dating them, so I could use this to create my own interface to track my spending exactly they way I want. Integrating this with Google's Firebase I can add budgets to further aid in spending control.
