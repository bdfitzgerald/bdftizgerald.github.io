Guild Wars 2 App

Project Details

Guild Wars 2 is a massively multiplayer online game built by Arena Net. The game has been running for several years but still does not have its own companion app. Arena net provide an API to access key in game information as well as an OAUTH2 account login to access personal account information relating to the game. Using ionic and angular I am creating a companion app based on arena nets API to allow users to get quick information easily whilst playing teh game, or to check up on their in game characters when they are not at home.
