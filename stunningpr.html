Project Details

Stunning PR is a simple WordPress website I built for a client. They required a simple site to advertise their affiliate links, as well as links to their other sites. As content was minimal I reflected that with a minimalistic one page website, matching the colour scheme to the existing logo. Based on google materials design the site adheres to modern design principles to provide a user friendly experience. By using WordPress the client is able to update affiliate links when an if needed providing them with a flexible solution to their needs.
