Lion King

Project Details

This site was a University project to showcase basic HTML5 and CSS3 to create a semantically built website. The main aims where to design a informative site with easy to reach information about the site topic. I chose the Lion King as there were already a large number of usable design assets available as well as having a vibrant color scheme. This meant I could focus on creating a flexible and semantic site to showcase the skills I had learned instead of spending time creating a design pattern to be used. The vast amount of assets meant I was able to incorporate a variety of methods to display this such as modals, video and images tags. I was also able to incorporate CSS3 properties to display the assets in a more viewable manner, such as the blur on the header image.
